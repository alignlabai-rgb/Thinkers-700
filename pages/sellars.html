「見たまま」は嘘

「私は赤いリンゴを見ている」——この単純な知覚報告は、哲学的に何を意味するでしょうか。伝統的経験論は、こうした感覚経験が知識の「土台」——それ自体で確実であり、他のすべての知識の基盤となる「所与（the Given）」——だと考えてきました。ウィルフリド・セラーズは1956年の論文「経験論と心の哲学」で、この「所与の神話」を徹底的に批判しました。

セラーズによれば、「赤いものを見ている」という報告ですら、概念的枠組みに依存しています。「赤」という概念を持っていなければ、私たちはリンゴを「赤い」とは認識できません。知覚は「概念的に負荷されている」のです。知識には「直接与えられる」土台はなく、すべての認識は「理由の空間」——概念間の推論的関係のネットワーク——の中で成立する。

セラーズを学ぶことは、「純粋な経験」「直接的知覚」という素朴な考えを問い直すことです。AI、認知科学、科学哲学——「表象」や「知識」とは何かを問うあらゆる分野で、セラーズの洞察は今も根本的な問いを突きつけています。

💡 たとえるなら

セラーズは、「知識に『生データ』は存在しない。すべての入力はすでに前処理（概念化）されている。『所与』とは、まるでOSを介さずにハードウェアに直接アクセスできるかのような幻想だ」と主張した、認識論のミドルウェア理論家です。彼は「直接知覚」というAPIは存在せず、すべてが概念フレームワークというランタイム上でのみ動作することを示しました。

📐 理由の空間

知識は因果関係ではなく「理由」の問題。信念は他の信念との推論的関係によって正当化される。この規範的ネットワークが「理由の空間」。

🔬 科学的イメージ vs 顕在的イメージ

日常的世界像（色、音、固体）と科学的世界像（原子、電磁波）の対立。両者をいかに統合するかが哲学の課題。

🧠 心理学的唯名論

心的状態（信念、欲求など）は内的エピソードではなく、行動傾向と推論的役割によって規定される。「内観」の特権性を否定。

🗣️ 言語共同体

概念は言語共同体の中で獲得される。「赤い」の意味は、その使用規則（いつ「赤い」と言うか、何が帰結するか）によって決まる。

🎭 機能的役割意味論

概念の意味はその推論的役割——他の概念との関係、行為への帰結——によって構成される。意味は「頭の中」にはない。

🔄 所与から理由の空間へ

1. 伝統的経験論

感覚印象（sense impressions）が知識の土台。これが「与えられる」（given）ものであり、解釈や推論に先立つ。確実性の源泉。

2. 神話の暴露

セラーズは「所与」を批判。感覚状態が「知識」であるためには、それが概念的に分節化されていなければならない。純粋な所与は「知識」ではない。

3. 概念的負荷

「赤いリンゴを見ている」という報告は、すでに「赤」「リンゴ」「見る」という概念を前提としている。知覚は概念的に負荷されている。

4. 理由の空間

知識は「理由の空間」——概念間の推論的・規範的関係のネットワーク——の中でのみ成立する。孤立した「所与」は知識にならない。

💻 System Analysis: No Raw Data Access

Myth of Given as Raw Pointer Illusion

「所与の神話」は、OSを介さずにハードウェアに直接アクセスできるという幻想に似ている。実際にはすべてのデータアクセスはドライバ（概念フレームワーク）を経由する。「生データ」は存在しない。

Space of Reasons as Type System

理由の空間は強い型システムに似ている。命題は他の命題との関係（型の互換性、推論規則）によって「意味」を持つ。孤立した値に意味はない。

Conceptual Loading as Preprocessing

すべての入力は前処理（概念化）される。センサー入力（感覚）はすでにパーサー（概念枠組み）を通過してからアプリケーション層（意識）に届く。

Manifest vs Scientific Image as UI vs Backend

顕在的イメージはユーザーインターフェース、科学的イメージはバックエンド。ユーザーは「ボタン」を見るが、実際には「関数呼び出し」が起きている。両者の整合性が課題。

🔮 思考実験：色盲者と「赤」

あなたの「赤」は私の「赤」か

メアリーは生まれつき色盲で、白黒の世界で育ちました。しかし彼女は色彩科学の専門家であり、「赤」についてあらゆる科学的事実——波長、網膜の反応、脳の処理——を知っています。

ある日、メアリーは治療を受けて色が見えるようになります。初めて赤いトマトを見たとき、彼女は新しい何かを「学んだ」のでしょうか。

この「メアリーの部屋」（フランク・ジャクソンの思考実験）に対して、セラーズ的視点はどう応答するでしょうか。セラーズなら、メアリーが新たに獲得したのは「事実」ではなく、「赤い」という概念の新たな推論的役割——つまり、「この感覚状態を『赤い』と報告する」能力——だと言うかもしれません。

「赤の経験」は、概念的枠組みの中での位置づけなしには、知識とはなりえないのです。

📝 結論

ウィルフリド・セラーズの「所与の神話」批判は、20世紀後半の認識論・心の哲学に革命をもたらしました。「純粋経験」が知識の土台であるという経験論の根本前提を覆し、すべての認識が概念的に負荷されていることを示しました。「理由の空間」という概念は、知識を因果ではなく規範的・推論的関係として再定義し、ブランダムやマクダウェルの推論主義に直接影響を与えました。AIが「知覚」や「理解」を行うと言われる現代において、「概念なしの認識は可能か」「機械は理由の空間に参加できるか」というセラーズの問いは、かつてないほど実践的な意味を持っています。
